## 功能特性

- 命令行交互界面，页面加载后自动聚焦，直接输入即可
- 命令历史 (↑/↓ 键)
- Tab 自动补全
- 支持多种命令：`about`、`skills`、`projects`、`contact`、`neofetch` 等
- 机械键盘按键音效（Web Audio API）
- `hello world` 彩蛋：ASCII Art 逐行打字动画
- 右下角半透明虚拟键盘 & 鼠标可视化，实时响应物理输入
- 自定义块状闪烁光标，跟随输入内容移动

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 自定义内容

编辑 `src/commands/index.ts` 中的常量来自定义个人信息：

- `ABOUT` - 个人介绍
- `SKILLS` - 技术栈
- `PROJECTS` - 项目经历
- `CONTACT` - 联系方式

## 技术栈

- React 18
- TypeScript
- Vite
