## 功能特性

- 命令行交互界面
- 命令历史 (↑/↓ 键)
- Tab 自动补全
- 支持多种命令：`about`、`skills`、`projects`、`contact`、`neofetch` 等

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 自定义内容

编辑 `src/commands/index.ts` 中的常量来自定义个人信息：

- `ABOUT` - 个人介绍
- `SKILLS` - 技术栈
- `PROJECTS` - 项目经历
- `CONTACT` - 联系方式

## 技术栈

- React 18
- TypeScript
- Vite
